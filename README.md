# 📊 Customer-Churn-Prediction 
## 📜 Overview 
Machine Learning Project to predict Customer Churn Prediction using data analysis ,feature Engineering and classification models.
## 🔮 Models and Accuracy 
- Logistic Regression: 73.8%
- Random Forest: 78.63%
## 📊 Exploratory Data Analysis
- Performed Univariate and Bivariate Analysis
- Analysed impact of features like tenure, monthly charges and contract type on churn
## 🎨 Feature Engineering
- Handling Missing values
- Encoded Categorical Variables
- Applied Standard Scaling for Logistic Regression
- Handled class imbalance using SMOTE
##  📈 Results
Best Model:Random Forest
Identified important features using feature importance
